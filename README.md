# 🎂 Happy 22nd Birthday, Moumita! 💖

A beautiful, interactive birthday website created with love for Moumita's 22nd birthday!

## 🌸 Features

✨ **Welcome Section** - Warm greeting with floating hearts animation

📸 **Memories Gallery** - Interactive photo gallery with smooth transitions

💌 **Letter from Brother** - Heartfelt message from Bikash

✨ **22 Wishes for 22 Years** - Interactive wish cards that reveal one at a time

🎁 **Surprise Button** - Confetti animation and special birthday message

🎈 **Beautiful Design** - Soft pink, lavender, and gold theme

📱 **Mobile-Friendly** - Fully responsive design for all devices

## 🎨 Design Theme

- **Colors**: Soft pink (#FFB6D9), Lavender (#E6D9F5), Gold (#FFD700)
- **Animations**: Smooth transitions, floating effects, and interactive elements
- **Typography**: Modern, clean, and readable fonts

## 📝 How to Use

1. Open `index.html` in a web browser
2. Scroll through the different sections
3. Click on the gallery arrows to browse through photos
4. Click on wish buttons to reveal all 22 wishes
5. Click the surprise button for a special moment!

## 📸 Customization

To add your own photos to the gallery:
1. Replace the placeholder image URLs in the gallery section
2. Or upload images to a cloud service and link them
